# Example of using typeorm/typescript

## Installation
1. Clone respository `git clone https://github.com/alwspiderc/node_crud.git`
2. `cd node_crud`
3. run yarn
4. edit in `env.` your database

## Run project 
`yarn dev` or `npm install`

## Create migration
`yarn migration:generate`

## Run migrate
`yarn migration:run`
